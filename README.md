Evtree is a Go library that computes a deterministic, directory-aware Merkle tree hash over a set of files. Given a list of file entries, each described by a relative path, byte size, and SHA-256 digest, the library reconstructs the directory hierarchy from the file paths and recursively hashes each directory node. Leaf nodes are constructed by prepending a domain separation byte (0x00) to a canonical string representation of the file metadata, then computing the SHA-256 digest. Internal directory nodes are computed by sorting their children lexicographically by name, concatenating the child hashes with a distinct domain separation byte (0x01), and hashing the result. This structure ensures that the final root hash is sensitive to both the content and the hierarchical organization of the file tree, and that it is fully deterministic regardless of the order in which file entries are provided. The library is primarily used as a dependency of [androidqf](https://github.com/mvt-project/androidqf), [MESH](https://github.com/BARGHEST-ngo/MESH), and WARD, where it provides tamper-evident integrity verification of acquired forensic artifacts.
